# Color Scheme - Light Mode

This document outlines the color scheme used in light mode for the application, based on the CSS custom properties defined in `app/assets/css/main.css`.

## Primary Colors

- **Background**: `hsl(45 25% 96%)` - A very light cream/beige background color
- **Foreground**: `hsl(340 60% 30%)` - A muted reddish-brown text color
- **Border**: `hsl(30 20% 85%)` - A light orange-tan border color
- **Input**: `hsl(30 20% 85%)` - Same as border, used for input fields
- **Ring**: `hsl(340 55% 45%)` - A medium reddish-brown for focus rings

## Additional Notes

- Colors are defined using HSL (Hue, Saturation, Lightness) values for consistency and ease of theming.
- The application uses Tailwind CSS with custom CSS variables for theming.
- Dark mode variants are defined separately with different HSL values.
